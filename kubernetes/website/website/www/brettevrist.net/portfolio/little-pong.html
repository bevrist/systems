Little Pong 326 Lines of c++ Code

Description: This project was my first use of an external library in c++ (SFML). This project was used to learn about object collisions, and 2D rendering using the SFML library. This project was also used to learn how to create basic AI for use in a game.

My Role: Full Development, using the SFML library https://www.sfml-dev.org/

Functionality: A c++ SFML rendition of the classic game “pong”. Features 2 versions: normal mode and “psychedelic mode”, which has changing colors as the game progresses. A score counter for keeping track of play, the ball moves slightly faster the longer the game progresses. An AI opponent who attempts to always hit the ball back to the player, but is designed to sometimes miss. The application also has a “game over” screen which allows the user to see their high score and restart the game.

Technical Skills Demonstrated: C++ object interaction, event handling, and collision calculation.

Summary: In this project, I learned how to use the SFML library and use class communication between objects to calculate physics collisions, as well as learning how to work with events.
